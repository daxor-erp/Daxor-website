# Daxor ERP Website

AI-native enterprise resource planning platform with expert consulting, data migration, and corporate training services.

## Project info


## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui components

## Development

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Features

- AI-native ERP platform
- Cloud consulting (AWS, Azure, Snowflake)
- Data migration services (Informatica → Microsoft Fabric)
- Corporate training programs
- Interactive architecture visualization
- Responsive design with dark/light mode
